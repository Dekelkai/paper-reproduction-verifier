# 模板使用说明

## 文件清单

- `复现报告模板.tex` - LaTeX 报告模板（主模板）
- `复现报告示例.tex` - 完整示例（参考）
- `复现报告示例.pdf` - 示例 PDF（参考）

## 使用方式

### 1. 直接使用模板

读取 `复现报告模板.tex`，替换所有 `<...>` 占位符。

### 2. 编译命令

```bash
xelatex -interaction=nonstopmode 复现报告.tex
```

**要求**：
- XeLaTeX 编译器
- ctex 宏包（中文支持）

### 3. 占位符说明

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `<编号>` | 论文编号 | 520260326000002 |
| `<论文英文标题>` | 论文英文标题 | AI-Assisted Traditional Music... |
| `<论文中文名>` | 论文中文标题 | 基于 MFCC 声学特征学习... |
| `<随稿文件>` | 随稿文件清单 | 论文 docx / 代码 main.py / ... |
| `<完全复现/部分复现/无法复现>` | 复现结论 | 完全复现 |
| `<XX>` | 复现率百分比 | 99 |

### 4. 表格填写

#### 复现指标对比表
```latex
<指标1> & <论文值> & <复现值> & <差异> & <接近/不一致/缺失> \\
```

**示例**：
```latex
WeightedEnsemble\_L3 测试准确率 & 0.9484 & 0.9439 & -0.0045 & 接近 \\
```

#### 问题反馈
使用 `\issue{类别}{问题描述}{证据}{建议作者补充}` 命令。

**示例**：
```latex
\issue{数据集问题}{数据集无法获取，链接失效。}{README.md 提到需下载 XYZ 数据集，但链接 404。}{建议作者提供数据集镜像。}
```

**如果无阻断性问题**：
```latex
\section{问题反馈}
\noindent 本次复现未发现阻断性问题。代码可直接运行，核心结果高度一致。
```

### 5. 输出文件位置

生成在 `../复现报告/` 目录：
```
复现报告/
├── 复现报告-论文编号-简称.tex
└── 复现报告-论文编号-简称.pdf
```

## 模板结构

1. 基本信息（表格）
2. 复现环境（表格）
3. 复现过程记录
   - 论文工作的理解
   - 根据论文应复现的关键实验
   - 仓库与运行说明检查
   - 实际执行过程
   - 代码与论文的一致性分析
4. 评价指标
5. 复现指标对比（长表格）
6. 问题反馈
7. 结论
   - 分析
   - 结论
   - 复现率

## 注意事项

1. **保持模板结构不变**，内容精简但结构完整
2. **删除无意义占位符**：不要保留 `<...>` 在最终 PDF 中
3. **表格对齐**：使用 `&` 对齐列，`\\` 换行
4. **中文支持**：必须使用 XeLaTeX + ctex
5. **特殊字符转义**：`_` → `\_`, `%` → `\%`, `#` → `\#`
